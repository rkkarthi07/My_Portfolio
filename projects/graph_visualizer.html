Graph visualizer of Shortest Path - Dijkstra's algorithm

  • Tech Stack: HTML, CSS, Javascript
  • Website URL: Link

Implements a visual representation of a graph, allowing users to add nodes, create edges, and find the shortest path using Dijkstra's algorithm. It includes functions for adding nodes, creating edges, and finding the shortest path from a specified source node.

The shortest path between nodes in a weighted graph. It starts from an initial node and iteratively expands to neighboring nodes, updating their distances from the source. This process continues until all reachable nodes have been visited.
